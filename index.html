DDoS Attack & Pen-testing Tool-kit

Developed a comprehensive tool-kit to simulate Distributed Denial of Service (DDoS) attacks and conduct penetration testing audits, identifying security vulnerabilities in networks and web applications. Scripts were created to launch controlled DDoS and proxy attacks, testing the resilience of servers while hiding the source IP using proxies.

The tool-kit leveraged open-source utilities like SQLMap for SQL injection testing and Nmap for network scanning. It identified common vulnerabilities like directory traversal and exposed misconfigurations, providing detailed reports that helped developers improve the security and stability of their applications.
